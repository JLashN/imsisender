Este es un repositorio en el que he creado unos programas que usan un cliente y un servidor en python para enviar imsis procesados.
## Estructura
En la carpeta Code podemos encontrar el codigo que esta dividido en librerias, notebooks y scripts.
Los scripts es lo que se va a ejecutar en ultima instancia, los notebooks son para pruebas y las librerias son cosas necesarias para los scripts.
En la carpeta Keys iran guardadas las claves publica y privada.
En la carpeta Env esta el enviroment necesario de anaconda.
